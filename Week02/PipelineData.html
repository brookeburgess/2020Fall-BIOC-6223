Sample Virus Pipeline

  1. All data is stored on a HIVE instance on a linux server.
  2. Starting with file 1 and file 2, align the input file 1 to input file 2 using the Hexagon tool.
  3. This will produce several files. Take the alignment file and use Heptagon to call SNPs.
  4. Heptagon produces three files, all_indels.dict, FreqProfile.csv, and SNPprofile. All of the files are appended to the base URL https://hive2.biochemistry.gwu.edu/dna.cgi?cmd=objFile&ids=14782&filename=all_indels.dict.
